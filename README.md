# StudyPilot AI

Starter scaffold for StudyPilot AI — a Next.js + TypeScript + Tailwind application with Supabase and Stripe integration.

What I added:
- Next.js app router scaffold (app/)
- Authentication pages (signup, login, forgot password) using Supabase
- Dashboard layout, navbar, sidebar
- Course management pages skeleton
- API route placeholders for uploads, summaries, and Stripe webhooks
- Supabase SQL schema (supabase/schema.sql)
